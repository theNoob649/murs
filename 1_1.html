Minecraft 1.1: The Adventure Update

Release Date: January 12, 2012

The 1.1 update was mostly a bug-fix patch but also introduced some key features to improve gameplay.

  • Spawn Eggs: Added to creative mode, allowing players to spawn mobs instantly.
  • Added Structures: Villages got improvements with more varied buildings.
  • Performance Enhancements: Optimized game engine for smoother gameplay.
  • New Command: The /gamerule command introduced for customizing gameplay rules.
  • Texture Improvements: Several block textures and sounds updated for better immersion.

While not a huge content update, 1.1 polished Minecraft and set the stage for bigger features ahead.
